# 👪 우리.zip
# 가족 간 소중한 순간을 기록하고 공유하는 AI 기반 서비스

우리.zip은 가족 구성원들이 일상 사진·영상, 주차별 영상 답변, 반려동물 기록 등을 한 공간에 모아 공유할 수 있도록 제작된 서비스입니다.  AI 기반 자동 분류, 영상 요약, 웃는 썸네일 생성 등 다양한 기술을 활용하여 가족 간 소통을 더욱 쉽고 즐겁게 만들어 줍니다.

## 🧩 Repository Structure

| 분야 | 레포지토리 | 설명 |
|------|-------------|------|
| Frontend | WooriZip-FE | React 기반 웹앱 구현, UI/UX, 상태관리, API 연동 |
| Backend | WooriZip-BE | Spring Boot 기반 서버 API, 인증/인가, DB 설계 및 관리 |
| AI Service | WooriZip-AI | 썸네일 자동 생성, 얼굴 위치 분석, 반려동물 감지 및 쇼츠 추출, STT 기반 요약 |

---

## ✨ Key Features

- 주차별 질문 제공 및 영상 답변 업로드  
- AI 기반 자동 썸네일 생성, 제목/내용 요약, 얼굴 위치 안내  
- 반려동물 등장 구간 자동 감지 및 쇼츠 생성  
- 일상 기록(사진/영상) 업로드 및 댓글 소통 기능  
- 멤버별 / 반려동물별 자동 기록 분류 보관함  
- 가족 간 정서적 소통을 위한 1:1 쪽지함  
- TTS 음성 설정, 글자 크기 조절 등 사용자 편의 기능 제공
